NoiseCapture

Download

All data produced by the NoiseCapture community are available at this page. Data are free and delivered under the ODbL terms.

Information

One .zip file is given for each country where measures have been made. The .zip files are nightly updated (3h30 AM UTC) and contains a README.html file for the description of the data and .geojson files for each region covered by measurements.

Cite NoiseCapture data

If you want to cite NoiseCapture in a scientific paper (or else), please refer to this article:
Bocher et al. 2017, Collaborative noise data collected from smartphones, Data in Brief 14C (2017) pp. 498-503
